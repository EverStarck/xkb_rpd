# Hyprland configuration

1. Add `rpd` layout inside the path `.config/xkb/symbols`
2. Add `rpd` to hyprland config:
```bash
input {
    kb_layout = us, rpd
    ...
}
```

To change the layout use `hyprctl keyword input:kb_layout rpd`
